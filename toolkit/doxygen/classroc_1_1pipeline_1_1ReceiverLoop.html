Detailed Description

Receiver pipeline loop.

This class acts as a task-based facade for the receiver pipeline subsystem of roc_pipeline module (ReceiverSource, ReceiverSlot, ReceiverEndpoint, ReceiverSessionGroup, ReceiverSession).

It provides two interfaces:

  • sndio::ISource - can be used to retrieve samples from the pipeline (should be used from sndio thread)
  • PipelineLoop - can be used to schedule tasks on the pipeline (can be used from any thread)
Note
Private inheritance from ISource is used to decorate actual implementation of ISource - ReceiverSource, in order to integrate it with PipelineLoop.
