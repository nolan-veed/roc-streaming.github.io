Contacts¶

GitHub¶

We use GitHub for bug tracking and code review. Feel free to file an issue to report a bug, suggest a feature, or discuss an idea. Pull requests are more than welcome! See Contribution guidelines.

There is also a forum and Q&A board on GitHub discussions. Welcome!

Mailing list¶

We have an announcement and discussion mailing list for users and developers: roc@freelists.org. It is the recommended place for extensive discussions and troubleshooting.

You can browse the list archive and join the list by visiting this page. You can also join the list by sending an email to roc-request@freelists.org with “subscribe” in the “Subject” field.

There is also an RSS feed for the list.

Note

You need to join to be able to post.

Note

Please use bottom-posting on the mailing list.

Matrix chats¶

We also have matrix rooms for more interactive discussion:

  • #roc-streaming:matrix.org - user chat (web link)

  • #roc-streaming-dev:matrix.org - developer chat (web link)

You can open the chat right in your browser using web links above, or you can install Element (a popular matrix client) to desktop or mobile device. Except Element, there are plenty of other matrix clients.

Note

You need to setup Matrix account to be able to join the room.
